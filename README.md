# Sistema de Conta em Python

Projeto simples de estudo usando classes em Python para representar contas bancarias, clientes e extrato.

## Arquivos

- `Cliente.py`: classe com os dados do cliente.
- `Conta.py`: classe principal de conta, com deposito, saque, transferencia e saldo.
- `Extrato.py`: classe responsavel por armazenar e exibir transacoes.
- `ContaClienteExtrato.py`: exemplo de composicao entre conta e extrato.
- `Agregacao.py`: exemplo de agregacao entre conta e clientes.

## Como executar

```bash
python Agregacao.py
```

Ou, para testar a versao com extrato, importe/crie objetos usando as classes `Conta`, `Cliente` e `Extrato`.
