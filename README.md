# 🏦 Sistema de Conta Bancária em Python

Sistema bancário simples desenvolvido em Python com operações básicas realizadas diretamente pelo terminal.

---

## ✨ Funcionalidades

- 💰 Realizar depósitos
- 💸 Realizar saques
- 📄 Consultar saldo
- 🔒 Validação de operações
- 🖥 Interface simples no terminal
